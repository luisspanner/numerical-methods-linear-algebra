# Numerical Methods – Linear Algebra & Function Approximation

Implementations of numerical methods developed alongside the course 
"Linear Methods in AI" (2nd Semester, OTH Regensburg).

## Implemented Methods

- **Gaussian Elimination** – forward elimination with partial pivoting and back substitution
- **LR Decomposition** – with partial pivoting and permutation vector
- **Cholesky Factorization** – for symmetric positive definite matrices
- **Legendre Polynomial Approximation** – orthonormal basis via recursion, 
  variable transformation, and least-squares coefficient computation

Implementations cover core cases but are not fully generalized.

## Tech Stack

Python, NumPy, SciPy, Matplotlib
